# 3Dモードのセルウィンドウタイトルをcell-1形式に変更

## 概要

3Dモードで表示するセルウィンドウのタイトルを、DEPENDENCIESパネルと同じように`cell-1`, `cell-2`の形式で表示するように変更します。

## 実装内容

### 変更ファイル

- [`src/components/editor/renderers/cell-3d-wrapper.tsx`](src/components/editor/renderers/cell-3d-wrapper.tsx)

### 実装手順

1. **必要なインポートを追加**

- `displayCellName`関数を`@/core/cells/names`からインポート
- `useCellIds`フックを`@/core/cells/cells`からインポート

2. **cell-3d-wrapper.tsxの修正**

- 現在の58行目付近の`cellName`の計算ロジックを変更
- `useCellIds()`を使って`inOrderIds`を取得
- `cellIds.inOrderIds.indexOf(cellId)`でセルのインデックスを取得
- `displayCellName(cellData?.name || "_", cellIndex)`でタイトルを生成

### 実装詳細

DEPENDENCIESパネルの`custom-node.tsx`（38行目、73行目）と同じロジックを使用：

```typescript
const cellIds = useCellIds();
const cellIndex = cellIds.inOrderIds.indexOf(cellId);
const cellName = displayCellName(cellData?.name || "_", cellIndex);
```
