# 3Dモ

ードグリッド設定項目削除

## 概要

3Dモードのグリッド設定から以下の5項目を削除し、2Dモードと同様に固定値として扱う：

- Grid Spacing X（X方向の間隔）
- Grid Depth（Z方向の間隔、`spacingZ`）
- Grid Height（Y方向の高さ、`spacingY`）
- Grid Opacity（グリッド平面の透明度、0-1）
- Snap to Grid（グリッドにスナップするか）

## 変更ファイル

### 1. [src/components/editor/renderers/3d-layout/types.ts](src/components/editor/renderers/3d-layout/types.ts)

- `Grid3DConfig`インターフェースから`spacingX`, `spacingY`, `spacingZ`, `gridOpacity`, `snapToGrid`を削除
- `DEFAULT_GRID_3D_CONFIG`からこれらのデフォルト値を削除

### 2. [src/components/editor/renderers/3d-layout/grid-3d-controls.tsx](src/components/editor/renderers/3d-layout/grid-3d-controls.tsx)

- UIから以下の5つのコントロール要素を削除：
- Grid Spacing X（129-144行目）
- Grid Depth（145-161行目）
- Grid Height（162-178行目）
- Grid Opacity（179-197行目）
- Snap to Grid（198-212行目）

### 3. [src/components/editor/renderers/3d-layout/__tests__/grid-3d-controls.test.tsx](src/components/editor/renderers/3d-layout/__tests__/grid-3d-controls.test.tsx)

- 削除された項目に関するテストを削除または修正：
- `should render all controls`テスト（15-34行目）から、`spacing-x-input`, `spacing-z-input`, `spacing-y-input`, `opacity-input`, `snap-switch`の存在確認を削除
- `should have NumberField inputs`テスト（39-56行目）から、Grid Spacing X, Grid Depth, Grid Height, Grid OpacityのLabel確認を削除
- `snapToGrid`スイッチのテスト（93-108行目）を削除
- `should display current config values`テスト（110-131行目）から、`spacingX`と`gridOpacity`の使用を削除

### 4. [src/components/editor/renderers/3d-layout/__tests__/grid-3d-config-integration.test.tsx](src/components/editor/renderers/3d-layout/__tests__/grid-3d-config-integration.test.tsx)
