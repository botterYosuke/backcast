# Backcast Manual

Backcast は、戦略を書き、Replay で検証し、Live / Auto へ進めるためのデスクトップアプリです。

このサイトは製品利用者向けの操作マニュアルを正面入口にします。設計判断や実装記録は「技術者向け」に置きます。

## 読む順番

戦略の書き方を学びたい人は、まず**チュートリアル**から。すぐ動かしたい人は**サンプル戦略**を
コピーしてください。アプリの操作手順は**ユーザーマニュアル**にまとめています。

1. [チュートリアル](tutorial/index.md) — 戦略の書き方を段階的に学ぶ
2. [サンプル戦略](samples/index.md) — コピーして動かせる戦略の見本帳
3. [インストールと起動](manual/install.md)
4. [Replay を実行する](manual/replay.md)
5. [Live / Auto を使う](manual/live-auto.md)
6. [トラブルシュート](manual/troubleshooting.md)

## ドキュメントの正本

利用者向けの操作手順はこのマニュアルを正とします。

技術者向けの詳細仕様は、原則としてソースコードとテストを正とします。ADR は設計意図と制約、findings は実装時の判断記録です。
