# ユーザーマニュアル

この章は Backcast を使って戦略を検証、運用する人のための手順です。

## 最小ワークフロー

1. アプリを起動する。
2. Strategy Editor で戦略ファイルを開く、または新規作成する。
3. Replay の対象期間、足種、銘柄、初期資金を設定する。
4. Run で Replay を実行する。
5. チャート、注文、ポジション、買付余力、実行結果を確認する。
6. 必要に応じて Live / Auto へ進む。

## 注意

配布版では、同梱 Python runtime をアプリが自動で使います。利用者が別途 Python 環境を用意する必要はありません。
