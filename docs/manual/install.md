# インストールと起動

## 配布版を使う

1. 配布された Backcast インストーラーを起動します。
2. 画面の案内に従ってインストール先を選びます。
3. インストールが完了したら、スタートメニューまたはデスクトップショートカットから Backcast を起動します。
4. 初回起動後、ワークスペース画面が表示されることを確認します。

インストーラー版には Python runtime が同梱されます。別途 Python をインストールする必要はありません。

## 初回起動で確認すること

- Backcast のメイン画面が開くこと
- Strategy Editor が表示されること
- Replay 設定を入力できること
- Universe に銘柄を追加できること

## アンインストール

OS のアプリ管理画面から Backcast をアンインストールします。保存済みの戦略ファイルや市場データを削除する場合は、アンインストール後にユーザーデータの保存場所も確認してください。
