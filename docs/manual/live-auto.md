# Live / Auto を使う

Live / Auto は実際の接続先を使うため、Replay より慎重に扱います。

## 事前確認

1. 接続先の取引ツール、または API サーバーが起動していることを確認します。
2. 認証情報を入力します。
3. 接続状態、買付余力、銘柄情報が表示されることを確認します。
4. Auto を開始する前に Replay の結果を確認します。

## 停止

Auto 実行中に停止する場合は、アプリの停止操作を使います。終了時には実行中の注文、接続、Python runtime が後始末されます。

## 注意

Live / Auto は接続先の状態に依存します。ログアウト、API 無効化、通信断、取引時間外などは実行結果に影響します。
