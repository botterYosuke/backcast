# Replay を実行する

Replay は、保存済みの市場データを使って戦略を検証するモードです。

## 手順

1. Strategy Editor で戦略ファイルを開きます。
2. Replay 設定で開始日、終了日、足種、初期資金を入力します。
3. Universe で検証対象の銘柄を追加します。
4. Run を押します。
5. 実行中はチャート、注文、ポジション、買付余力、実行結果を確認します。

## 銘柄の扱い

Replay の銘柄候補は、設定した終了日時点のローカルカタログから取得します。終了日が未設定の場合、候補を確定できません。

## 実行結果

Replay が完走すると、少なくとも次の情報を確認します。

- チャート更新
- 注文履歴
- ポジション
- 買付余力
- 実行ステータス

## チャート操作

- **ドラッグ**: 左ボタンで左右にドラッグすると視窓を時間軸方向にパン（過去/未来）できます。ドラッグ中は自動スケールが解除されます。
- **ホイール**: マウスホイールでカーソル位置を中心にズームします（cell_width を変更）。Ctrl+ホイールは全体ズーム（PanCam）に流れ、チャート自体はズームしません。
- **ダブルクリック**: 視窓をリセットし、最新バーが右端に来る既定状態 + 自動スケールへ戻ります。
- **ホバー（マウスオーバー）**: 視窓内にカーソルを置くとクロスヘアが表示され、右の板（ladder）の対応する行が強調表示されます。
- **板差分の強調**: 板の数量変化は対象行を 300ms 強調表示してから徐々に消えます（増加=色つき・減少/消滅=グレー）。

## ウィンドウ操作

- **タイトルバー・ダブルクリック**: その窓を viewport 中心へ据え、窓全体が入る最大ズーム（contain-fit、6% 余白）に 200ms ease-out でフレーミングします（issues #166 / #167、findings 0123）。全窓共通（チャート・板・買付余力・ポジション・注文・実行結果・Strategy Editor・Order・HITL）。drag 中・単クリック・右/中クリックでは発火しません。チャート本体のダブルクリック（視窓リセット）とは別ジェスチャです（チャート本体＝視窓を初期状態に戻す／タイトルバー＝その窓を viewport に framing する）。
- **Universe sidebar 行クリック**: 銘柄行をクリックすると、その銘柄の `chart:<iid>` 窓があれば同じフレーミング動作で viewport にセンタリングします（issue #168、findings 0123 §3）。窓が未生成・非表示のときは何もしません。Replay / Live どちらのモードでも動作します。

## 走らない場合

戦略ファイルが未ロードの場合、Run は開始されません。まず Strategy Editor で対象の `.py` ファイルを開いてください。
