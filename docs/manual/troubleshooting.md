# トラブルシュート

## Run が押せない、または開始されない

- Strategy Editor で戦略ファイルを開いてください。
- Replay の開始日、終了日、足種、初期資金を確認してください。
- Universe に銘柄が入っていることを確認してください。

## 銘柄候補が出ない

- Replay では終了日を設定してください。
- ローカルの市場データカタログが存在するか確認してください。

## 戦略の保存結果が期待と違う

- Save / Save As の対象ファイルを確認してください。
- 同名の sidecar がある場合、Replay 設定は sidecar 側が優先されることがあります。

## Live 接続できない

- 取引ツール本体が起動していることを確認してください。
- API 設定が有効であることを確認してください。
- 認証情報と環境設定を確認してください。

## 配布版で Python が起動しない

配布物に同梱 Python runtime が含まれているか確認してください。インストール済みの配布版で起きる場合は、ビルドパッケージングの問題として扱います。
